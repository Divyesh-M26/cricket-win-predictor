\# 🏏 Cricket Win Predictor AI



A machine learning project that predicts cricket match winning probability using ball-by-ball data.



\## Features



\- Ball-by-ball cricket analysis

\- Calculates:

&#x20; - Runs left

&#x20; - Balls left

&#x20; - Wickets left

&#x20; - Current run rate

&#x20; - Required run rate

\- Machine learning prediction

\- Streamlit web interface





\## Tech Stack



\- Python

\- Pandas

\- Scikit-learn

\- Streamlit

\- Logistic Regression





\## How to Run



Install:



pip install pandas scikit-learn streamlit joblib





Run:



python -m streamlit run app.py





\## Model



Algorithm:

Logistic Regression



Dataset:

Cricket ball-by-ball data
